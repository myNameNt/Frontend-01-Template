# 每周总结可以写在这里
## 有限状态机
###  -  每一个状态都是一个机器  
    - 在每一个状态里可以进行计算，存储，输出
    - 所有这些机器接受的输入都是一致的
    - 状态机的每一个机器本身没有状态，如果我们用函数 来表示的话，它应该是纯函数（无副作用）
### - 每一个机器知道下一个状态
    - 每一个机器都有确定的下一个状态
    - 每一个机器根据输入决定下一个状态
### js中的有限状态机（Mealy-人名）
>> 每个函数是一个状态
  function state(input) { // 函数参数就是输入
    // 在函数中可以自由地编写代码，处理每个状态的逻辑
    return next; // 返回值作为下一个状态
  }
